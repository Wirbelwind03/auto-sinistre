## Description

AutoSinitre est une application web full-stack permettant de gérér les sinistres liées au véhicules d'un client.

## Tech Stack
- Frontend: Vue, CSS
- Backend: Spring Boot  
- Base de données: PostgreSQL

## Installation 

#### 1. Clone the Repository
```bash
git clone https://github.com/Wirbelwind03/auto-sinistre.git
```


#### 2. Configurer les informations d'identifications (`application.yaml`)
``` yaml
spring:
  datasource:
    url: BDD_URL
    username: BDD_USERNAME
    password: BDD_PASSWORD
```
